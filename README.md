# Future Green — Official Website

Live site: [futuregreenindia.com](https://futuregreenindia.com)

A responsive, multi-page marketing website built for **Future Green**, an LED lighting manufacturer based in Bhopal, Madhya Pradesh, India.

## About the Project

Future Green manufactures LED lighting solutions for residential, commercial, industrial, and outdoor/landscape applications (COB spotlights, panel lights, flood lights, street lights, rope lights, and more). This site serves as their digital storefront — showcasing product categories, generating dealer/bulk enquiries, and providing easy contact options.

## Features

- **Responsive design** — optimized for desktop, tablet, and mobile
- **Product catalog** with category-based filtering (Architectural, Indoor, Landscape, Outdoor, Office, Commercial & Industrial, Stadium)
- **Contact form** with validation
- **WhatsApp click-to-chat integration**
- **Embedded Google Maps** for facility location/directions
- **SEO & social meta tags** (Open Graph, Twitter Cards) for better sharing and discoverability
- **Social media links** (Instagram, Facebook, LinkedIn)

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Pages

- `index.html` — Home page with hero, categories, about, contact, and social sections
- `products.html` — Full product listing with category query-param filtering
